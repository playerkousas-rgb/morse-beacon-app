# SKW Morse V3.0 — 升級說明與部署指南

這是你的摩斯密碼網頁 App 的升級版。具備**接收解碼（聲音 + 光源）**、**可安裝 PWA**，並在 V3.0 新增 Goertzel 精準鎖頻、解碼歷史、速查教學、音檔離線解碼、快速短語與主題字級設定。

---

## 一、功能總覽

| 功能 | 說明 |
|---|---|
| 🎙️ **聲音解碼（Goertzel）** | 用麥克風 **單頻演算法精準鎖 750Hz**，比 FFT 更準、更抗噪 |
| 💡 **光源解碼** | 用後鏡頭偵測畫面中央亮度變化，解出閃光摩斯碼 |
| 📊 **即時訊號表 + 自動門檻** | 動態追蹤背景雜訊與峰值，自動判 on/off；可手動調靈敏度 |
| ⏱️ **自動估算速度(WPM)** | 接收時自動估算對方發報速度 |
| 🕒 **解碼歷史** | 每次接收結束自動存檔（localStorage），可複製/刪除 |
| 📚 **速查表 / 教學** | 點 A–Z/0–9 以 750Hz 發聲學摩斯 |
| 📁 **音檔離線解碼** | 上傳含 750Hz 的錄音，離線解出內容 |
| ⚡ **快速短語** | SOS / OK / HELP / YES / NO / WAIT 一鍵填入 |
| 🎨 **主題與字級** | 深/淺色、字體大小，設定會記住 |
| 📱 **PWA 安裝** | `manifest.webmanifest` + `sw.js`，可「加到主畫面」離線運作 |
| 📑 **四分頁介面** | 發送 / 接收解碼 / 速查設定 / 說明安裝（原發送功能完整保留）|

> 注意：升級沒有新增檔案，全部功能都寫在 `index.html` 裡，所以**還是只要上傳同樣那 6 個檔案**即可。`sw.js` 的快取版本已更新為 v3.0.0，使用者重新整理會自動拿到新版。

## 三、部署到你的 GitHub + Vercel（最簡單流程）

### 方法 A：用 GitHub 網頁直接上傳（不需電腦指令，推薦）
1. 打開 https://github.com/playerkousas-rgb/morse-beacon-app
2. 點 **Add file → Upload files**
3. 把上面 6 個檔案一起拖進去（`index.html` 會覆蓋舊的）
4. 下方填 commit 訊息，例如 `升級 V2.0：新增聲音/光源解碼 + PWA`
5. 按 **Commit changes**
6. Vercel 已連動你的 repo，會**自動重新部署**，約 1 分鐘後 https://morse-beacon-app.vercel.app/ 就更新了

### 方法 B：用 Git 指令（如果你有 clone 到電腦）
```bash
# 在你的專案資料夾內，把新檔案複製進去後：
git add index.html manifest.webmanifest sw.js icon-192.png icon-512.png icon-512-maskable.png
git commit -m "升級 V2.0：新增聲音/光源解碼 + PWA"
git push origin main
```
push 後 Vercel 自動部署。

## 四、怎麼當「App」用（免上架、免收費）

部署後，用手機瀏覽器打開 https://morse-beacon-app.vercel.app/ ：

- **Android (Chrome)**：右上角 ⋮ → 「安裝應用程式 / 加到主畫面」，或點頁面右上「安裝」按鈕。
- **iPhone (Safari)**：底部「分享」→「加入主畫面」。

加完後桌面會出現圖示，點開就是全螢幕、無網址列，跟原生 App 幾乎一樣，而且**完全免費、不用上架**。

## 五、重要的瀏覽器限制（請務必知道）

這些是瀏覽器的安全規定，**任何網頁版都繞不過**，不是程式問題：

| 項目 | Android Chrome | iPhone Safari |
|---|---|---|
| 麥克風（聲音解碼） | ✅ | ✅ |
| 相機（光源解碼） | ✅ | ✅ |
| 螢幕閃光、蜂鳴、震動 | ✅ | ✅（震動部分機型不支援）|
| **實體閃光燈發送** | ⚠️ 部分機型支援 | ❌ 不支援 |

- 取用相機/麥克風**必須是 HTTPS**——你的 Vercel 網址本來就是 HTTPS，沒問題。
- 第一次按「開始接收」會跳出**權限請求**，要按允許。

## 六、使用解碼功能的小技巧

**聲音解碼**
- 在較安靜的環境效果最好。
- 兩支手機距離適中（10–50cm），對方用本 App 發送（750Hz）最準。
- 若一直誤判，把「靈敏度」滑桿往右（較嚴格）。

**光源解碼**
- 環境越暗越好，讓閃光點明顯。
- 把後鏡頭中央對準閃光來源（手電筒/螢幕閃光）。
- 同樣可用靈敏度滑桿微調。

兩種模式都會**自動估算發報速度**，所以對方用幾 WPM 都能解。
